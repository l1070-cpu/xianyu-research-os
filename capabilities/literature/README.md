# Literature Capability Pack｜文献能力包

## 适用阶段
课题立项、研究背景、Introduction、Discussion。

## 输入
- 关键词
- 疾病/机制/成分
- 时间范围
- 目标期刊方向

## 标准工作流
关键词设计 → PubMed/Web of Science/CNKI 检索 → 文献筛选 → 阅读摘要 → 提取方法 → 总结研究空白 → 形成文献笔记。

## AI 可以做
- 关键词扩展
- 文献总结
- 方法提取
- 研究空白分析
- Introduction/Discussion 素材整理

## 人必须做
- 判断文献质量
- 确认是否适合本课题
- 核对引用准确性

## 输出
- 文献笔记
- Research Gap
- Methods 摘录
- Key References
